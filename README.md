# html_to_pdf
html to pdf and upload to s3_bucket
==============================================================
- install python
- install pip
- install boto or boto3 
- install php 
- Learn little about AWS 
  - IAM Role and policy
  - S3 bucket
------------------------------------------------------------------------------------
  **How to Run Server**
- php -S 0.0.0.0:[port] 
- eg.- php -S 0.0.0.0:8080

------------------------------------------------------------------------------
If everythings goes you will be able to convert your HTML into Pdf and stor it to ASW S3 Bucket
